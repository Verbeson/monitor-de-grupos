Política de Privacidade

Última atualização: 12 de março de 2026

Esta política de privacidade descreve como a extensão Monitor de Grupos WhatsApp ("Extensão") trata os dados dos usuários.

1. Coleta de Dados

A Extensão não coleta, armazena ou transmite nenhum dado pessoal, mensagens ou conteúdo de conversas para servidores externos. Todo o processamento ocorre localmente no navegador do usuário.

2. Dados Armazenados Localmente

A Extensão utiliza a API chrome.storage.local para salvar as seguintes configurações no dispositivo do usuário:

  • Nomes de grupos monitorados
  • Nomes ou números de contatos ignorados
  • Palavras-chave ignoradas
  • Preferência de monitoramento ativo/inativo
  • Dados de ativação do plano Pro (email ou chave de licença)

Esses dados nunca saem do dispositivo do usuário, exceto a verificação de assinatura descrita abaixo.

3. Verificação de Assinatura (Plano Pro)

Quando o usuário opta pelo plano Pro e informa seu email, a Extensão envia apenas o email para nosso servidor (monitor-de-grupos.vercel.app) para verificar o status da assinatura no Mercado Pago. Nenhum outro dado é transmitido.

Essa verificação ocorre:

  • Quando o usuário ativa o plano Pro manualmente
  • A cada 24 horas para revalidação automática

4. Permissões Utilizadas

A Extensão solicita as seguintes permissões do Chrome:

  • storage: Para salvar configurações localmente no navegador
  • tabs: Para verificar se o WhatsApp Web está aberto e focar na aba
  • scripting: Para reinjetar o script de monitoramento quando necessário
  • Host permission (web.whatsapp.com): Para ler as notificações de mensagens na interface do WhatsApp Web

5. Acesso ao WhatsApp Web

A Extensão acessa a página do WhatsApp Web exclusivamente para:

  • Detectar novas mensagens na barra lateral de conversas
  • Extrair nome do grupo, remetente e texto da mensagem para filtragem
  • Exibir alertas sonoros e visuais conforme configuração do usuário

Nenhuma mensagem é copiada, armazenada ou enviada para qualquer servidor.

6. Compartilhamento de Dados

Não compartilhamos, vendemos ou transferimos dados de usuários para terceiros, em nenhuma circunstância.

7. Segurança

Como todos os dados são processados e armazenados localmente no navegador, a segurança dos dados está vinculada à segurança do dispositivo do usuário.

8. Direitos do Usuário

O usuário pode a qualquer momento:

  • Visualizar e modificar todas as configurações salvas pela Extensão
  • Desativar o plano Pro e remover o email cadastrado
  • Desinstalar a Extensão, o que remove todos os dados locais

9. Alterações nesta Política

Eventuais alterações serão publicadas nesta página com a data de atualização revisada.

10. Contato

Para dúvidas sobre privacidade, entre em contato pelo email: suporte@monitordegruposzap.com
