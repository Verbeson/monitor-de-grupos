Politica de Privacidade

Ultima atualizacao: 12 de marco de 2026

Esta politica de privacidade descreve como a extensao Monitor de Grupos WhatsApp ("Extensao") trata os dados dos usuarios.

1. Coleta de Dados

A Extensao nao coleta, armazena ou transmite nenhum dado pessoal, mensagens ou conteudo de conversas para servidores externos. Todo o processamento ocorre localmente no navegador do usuario.

2. Dados Armazenados Localmente

A Extensao utiliza a API chrome.storage.local para salvar as seguintes configuracoes no dispositivo do usuario:

  • Nomes de grupos monitorados
  • Nomes ou numeros de contatos ignorados
  • Palavras-chave ignoradas
  • Preferencia de monitoramento ativo/inativo
  • Dados de ativacao do plano Pro (email ou chave de licenca)

Esses dados nunca saem do dispositivo do usuario, exceto a verificacao de assinatura descrita abaixo.

3. Verificacao de Assinatura (Plano Pro)

Quando o usuario opta pelo plano Pro e informa seu email, a Extensao envia apenas o email para nosso servidor (monitor-de-grupos.vercel.app) para verificar o status da assinatura no Mercado Pago. Nenhum outro dado e transmitido.

Essa verificacao ocorre:

  • Quando o usuario ativa o plano Pro manualmente
  • A cada 24 horas para revalidacao automatica

4. Permissoes Utilizadas

A Extensao solicita as seguintes permissoes do Chrome:

  • storage: Para salvar configuracoes localmente no navegador
  • tabs: Para verificar se o WhatsApp Web esta aberto e focar na aba
  • scripting: Para reinjetar o script de monitoramento quando necessario
  • Host permission (web.whatsapp.com): Para ler as notificacoes de mensagens na interface do WhatsApp Web

5. Acesso ao WhatsApp Web

A Extensao acessa a pagina do WhatsApp Web exclusivamente para:

  • Detectar novas mensagens na barra lateral de conversas
  • Extrair nome do grupo, remetente e texto da mensagem para filtragem
  • Exibir alertas sonoros e visuais conforme configuracao do usuario

Nenhuma mensagem e copiada, armazenada ou enviada para qualquer servidor.

6. Compartilhamento de Dados

Nao compartilhamos, vendemos ou transferimos dados de usuarios para terceiros, em nenhuma circunstancia.

7. Seguranca

Como todos os dados sao processados e armazenados localmente no navegador, a seguranca dos dados esta vinculada a seguranca do dispositivo do usuario.

8. Direitos do Usuario

O usuario pode a qualquer momento:

  • Visualizar e modificar todas as configuracoes salvas pela Extensao
  • Desativar o plano Pro e remover o email cadastrado
  • Desinstalar a Extensao, o que remove todos os dados locais

9. Alteracoes nesta Politica

Eventuais alteracoes serao publicadas nesta pagina com a data de atualizacao revisada.

10. Contato

Para duvidas sobre privacidade, entre em contato pelo email: suporte@monitordegruposzap.com
